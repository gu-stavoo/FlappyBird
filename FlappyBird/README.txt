Neste repositório encontra-se o meu projeto do FlappyBird feito no Unity.

Tal projeto foi feito para a seleção de candidatos a vaga de desenvolvedor AR/ VR/ Games bolsista no Samsung Ocean Center de Manaus.

O tileset foi obtido através do seguinte link: https://www.spriters-resource.com/mobile/flappybird/sheet/59894/

A fonte foi obtida através do seguinte link: https://www.dafont.com/04b-19.font

O projeto aqui encontrado foi feito por mim (com auxílio de tutoriais disponíveis no YouTube).
